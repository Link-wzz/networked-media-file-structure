Process

First of all, I removed the background of 0-9 gesture and export them as png images

I finished the UI design in figma. To be honest, I was a little dissatisfied with the height of the hand on the page because the length of the arm was not enough to reach the center of the pc screen, so if I wanted to create the impression that the arm was sticking out from the edge of the screen, I had to place the image at the bottom of the screen. Also, to make the time transition to be very smooth, I didn't switch images directly. Instead I let the hands slide out from the bottom of the screen, then the new hand slowly rise up. On the left side of the image below is another page layout idea I thought of at first, that is to use a graphs to increase the distance between the hand and the edge of the screen, however such graphics make the page look less concise, also diminishes people's attention to the hand, I wanted the background to be as simple as possible, so that it would be easier for the user to concentrate on watching the hand change.

Next up is programming. Since my last project didn't adapt well to different screens, I was very careful this time. I used Flexbox and responsive layouts like vw and vh to make sure the web page was compatible with various screens. In fact, I even programmed the compatibility of mobile phones, and therefore this web page display perfectly on most mobile phones.
